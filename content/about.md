## About QubitLogic

QubitLogic is a technical blog covering **quantum computing**, **AI infrastructure**, and **Python development** — with a focus on honest benchmarks, reproducible code, and practical implementation over marketing copy.

Every article is written with working code that you can run. Benchmark numbers are reproducible. Opinions are based on evidence, not affiliate relationships.

---

## About the Author

**Stephane Patteux** is a software developer and AI infrastructure engineer based in the UK. He builds AI agent systems, backtesting pipelines, and quantum-classical hybrid applications.

**Background:**
- Full-stack Python development — FastAPI, LangChain, LangGraph, data pipelines
- Self-hosted AI infrastructure on Linux VPS (Nginx, systemd, CI/CD)
- Quantum computing with Qiskit — algorithms, circuit optimisation, hardware submissions
- Algorithmic trading systems and financial data infrastructure

QubitLogic documents the engineering decisions, benchmarks, and code that come out of building these systems — starting from the infrastructure layer (Phase 1) through quantum algorithm implementation (Phase 2) and into enterprise-grade security and tooling (Phase 3).

## The Series Structure

The site is organised as three linked phases:

| Phase | Section | Focus |
|---|---|---|
| 1 | [Infrastructure](/infrastructure/) | VPS provisioning, Nginx, Python, CI/CD |
| 2 | [Quantum Coding](/quantum-coding/) | Qiskit, QAOA, Grover, hybrid ML |
| 3 | [Professional Edge](/professional-edge/) | PQC, RAG agents, financial APIs, compliance |

Each phase builds on the previous one — the VPS from Phase 1 runs the quantum experiments in Phase 2, and the security tooling in Phase 3 protects the APIs built throughout.
