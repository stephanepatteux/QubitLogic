# Project Vision: QubitLogic.dev

## Objective
Build a high-performance, affiliate-focused technical blog dedicated to Quantum-AI Hybrid Development. The site serves as a "Technical Workbench" for developers and IT professionals. Domain: qubitlogic.dev

## Core Pillars
1. **Practicality:** Focus on code-first tutorials (Python, Qiskit, AI Agent APIs).
2. **Infrastructure:** Promote a "self-hosted" tech stack (DigitalOcean/Vultr VPS, Nginx, Linux).
3. **Monetization:** Passive, recurring revenue via high-intent affiliate programs (VPS hosting, AI SaaS platforms, specialized APIs).

## Technical Requirements
- **Generator:** Hugo (Extended).
- **Hosting:** Self-hosted Linux VPS (Ubuntu) via Nginx.
- **Design:** Minimalist, developer-centric, dark-mode default.
- **Performance:** Must load in < 500ms (Core Web Vitals optimized).
- **SEO/GEO:** Must be structured for AI search crawlers (Gemini, Perplexity, ChatGPT).

## Rules of Engagement
- **Authority:** Every technical recommendation must be based on data/benchmarking.
- **Affiliate Integration:** Links must be embedded contextually via reusable Hugo shortcodes (e.g., `{{< affiliate_box >}}`).
- **Safety:** No "get rich quick" content. Keep the tone professional, direct, and engineering-focused.
